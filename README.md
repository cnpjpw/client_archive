Uso: python carga\_db.py DD-MM-YYYY
baixa todos os dumps do archive do cnpjpw a partir da data informada e carrega no sqlite cnpjpw.db

Uso: python gerar\_json.py DD-MM-YYYY
gera json análogo ao formato do endpoint /cnpj/{cnpj} de todos os cnpjs da data passada presentes no banco local e salva no arquivo DD-MM-YYYY.json
